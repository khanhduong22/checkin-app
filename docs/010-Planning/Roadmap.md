# Product Roadmap

## Q1 2026: Core Foundations (Current)
- [x] **Attendance Tracking**: IP-based validation, check-in/out logic.
- [x] **Task Marketplace**: Admin posts, User claims, Payroll integration.
- [x] **Basic Gamification**: Lucky Wheel, Shop Pet (v1).
- [x] **Request System**: Leave/Late/WFH workflows.

## Q2 2026: Enhanced Engagement & Mobile
- [ ] **Mobile App (PWA)**: Optimized interactions for mobile, push notifications for task alerts.
- [ ] **Advanced Gamification**: Pet customization, Leaderboards for top performers.
- [ ] **Social Features**: Team challenges, "High Five" peer recognition.

## Q3 2026: Intelligence & Automation
- [ ] **Smart Scheduling**: Auto-assign shifts based on availability.
- [ ] **Payroll Analytics**: Trends analysis, budget forecasting for admins.
- [ ] **AI Assistant**: Chatbot for "How many leave days do I have?" type queries.

## Backlog / Future Ideas
- **External Integrations**: Slack/Discord notifications.
- **Biometric Auth**: Fingerprint/FaceID for mobile check-in.
- **Multi-Tenant**: Support for multiple organizations.
